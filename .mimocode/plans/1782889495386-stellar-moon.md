# 车间环境文件类型枚举方案

## 需求
在 `subTypeDirMap` 中添加车间环境子类型，更新 FileController 注释。

## 目录拼接逻辑
`getBusinessTypeDir("ROOM_CLEAN_INSPECTION_REPORT")` → split("_",2) → parent=`ROOM`, sub=`CLEAN_INSPECTION_REPORT` → `车间环境` + `洁净检测/报告`

## 一、修改 FileStorageConfig.java

在 `subTypeDirMap` 中添加：

```java
put("CLEAN_INSPECTION_REPORT", "洁净检测/报告");
put("CLEAN_INSPECTION_CERTIFICATE", "洁净检测/证书");
put("CLEAN_INSPECTION_PHOTO", "洁净检测/照片");
put("CLEAN_INSPECTION_ATTACHMENT", "洁净检测/附件");
put("TEMPERATURE_HUMIDITY_PHOTO", "温湿度记录/照片");
put("PRESSURE_DIFFERENCE_PHOTO", "压差记录/照片");
put("DISINFECTION_PHOTO", "消毒记录/照片");
```

## 二、更新 FileController.java 注释

```java
 * 车间环境（ROOM）：
 * - ROOM_CLEAN_INSPECTION              → 车间环境/洁净检测
 * - ROOM_CLEAN_INSPECTION_REPORT       → 车间环境/洁净检测/报告
 * - ROOM_CLEAN_INSPECTION_CERTIFICATE  → 车间环境/洁净检测/证书
 * - ROOM_CLEAN_INSPECTION_PHOTO        → 车间环境/洁净检测/照片
 * - ROOM_CLEAN_INSPECTION_ATTACHMENT   → 车间环境/洁净检测/附件
 * - ROOM_TEMPERATURE_HUMIDITY          → 车间环境/温湿度记录
 * - ROOM_TEMPERATURE_HUMIDITY_PHOTO    → 车间环境/温湿度记录/照片
 * - ROOM_PRESSURE_DIFFERENCE           → 车间环境/压差记录
 * - ROOM_PRESSURE_DIFFERENCE_PHOTO     → 车间环境/压差记录/照片
 * - ROOM_DISINFECTION                  → 车间环境/消毒记录
 * - ROOM_DISINFECTION_PHOTO            → 车间环境/消毒记录/照片
```
