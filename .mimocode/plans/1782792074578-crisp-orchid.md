# Plan: 接口 byPreparation 增加工序编码字段

## 背景
`/api/preparationProcessTemplate/byPreparation/{prepId}` 接口当前返回的 `PreparationProcessTemplate` 已有 `processTypeName`（工序类型名称），但缺少 `processCode`（工序编码）。工序编码存储在关联的 `process_type` 表中。

## 修改内容

### 1. `PreparationProcessTemplate.java` 添加 transient 字段
**文件**: `src/main/java/com/tonghui/erp/Data/Entity/PreparationProcessTemplate.java`

在现有 `processTypeName` 字段（line 125）附近添加：
```java
@TableField(exist = false)
private String processCode;
```

### 2. `PreparationProcessTemplateController.java` 填充 processCode
**文件**: `src/main/java/com/tonghui/erp/Controller/PreparationProcessTemplateController.java`

在 `fillNameFields()` 方法中（line 192-194），设置 `processCode`：
```java
if (processType != null) {
    template.setProcessTypeName(processType.getProcessName());
    template.setProcessCode(processType.getProcessCode());  // 新增
}
```

## 验证
```bash
./mvnw compile
```
编译通过即可，接口返回的 JSON 中每个元素会自动包含 `processCode` 字段。
