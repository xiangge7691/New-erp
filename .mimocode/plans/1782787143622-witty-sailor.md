# ERP 系统功能支持分析

| 模块 | 功能 | 状态 | 说明 |

| 用户信息 | 权限 | ✅ 已支持 | 完整 CRUD |
| 用户信息 | 角色 | ✅ 已支持 | CRUD + 权限分配 |
| 人员档案 | 部门 | ✅ 已支持 | 完整 CRUD |

| 人员档案 | 人员档案+证书 | ⚠️ 部分支持 | 有 CRUD + 健康证提醒，缺少证书上传和高级查询 |

| 产品信息 | 验证方案 | ✅ 已支持 | 通过 PreparationDocument 支持 + 文件上传 |

## 详细分析

### ✅ 已完全支持的功能

#### 1. 用户信息 - 账号/权限/角色
- **UserController** (`/api/User`): 完整 CRUD + 高级搜索（按用户名、部门、角色、状态）
- **AuthController** (`/api/auth`): 登录认证、JWT 令牌
- **PermissionController** (`/api/Permission`): 完整 CRUD
- **RoleController** (`/api/Role`): CRUD + 权限分配

#### 2. 人员档案 - 部门/岗位
- **DepartmentController** (`/api/Department`): CRUD + 模糊搜索
- **PositionController** (`/api/position`): CRUD + 关键词搜索 + 全量列表

#### 3. 设备档案 - 全部
- **RoomInfoController** (`/api/room`): CRUD + 按名称/位置搜索
- **EquipmentController** (`/api/equipment`): CRUD + 高级搜索（名称、型号、厂家、类型、房间、状态）
- **EquipmentMaintenanceController** (`/api/equipmentMaintenance`): CRUD + 维保提醒 + 维保周期设置

#### 4. 产品信息 - 全部
- **PreparationController** (`/api/preparation`): CRUD + 高级搜索
- **PreparationProcessTemplateController** (`/api/preparationProcessTemplate`): CRUD + 批量保存 + 按制剂查询
- **PreparationDocumentController** (`/api/preparationDocument`): CRUD + 按制剂/类型查询
- **FileController** (`/api/files`): 文件上传/下载/删除

---

### ⚠️ 部分支持（需要增强）的功能

#### 1. 生产管理 - 生产详情
**已有**: `ProductionProcessRecord` 实体包含 production_start, production_end, inspection_start, inspection_end 字段

**缺少**:
- 按时间范围筛选（下单时间、生产结束时间）
- 搜索接口不支持时间范围参数

**修改文件**:
- `Controller/ProductionProcessRecordController.java` — 添加时间范围筛选参数
- `Service/ProductionProcessRecordService.java` — 添加时间范围查询方法
- `Service/impl/ProductionProcessRecordServiceImpl.java` — 实现

#### 2. 订单管理 - 订单状态总览
**已有**: `ProductionPlan` 有 current_status 字段，有状态管理接口

**缺少**:
- 根据生产详情自动改变状态
- 按下单时间、生产结束时间筛选

**修改文件**:
- `Controller/ProductionPlanController.java` — 添加时间范围筛选到 search 接口
- `Service/impl/ProductionPlanServiceImpl.java` — 已有时间范围查询

#### 3. 首页 - 数据统计
**已有**: `DashboardController` 有 /summary 接口，包含生产统计、库存预警、审批统计

**缺少**:
- 预估产值统计
- 订单量统计
- 交付量统计
- 采购额统计
- 待交付量统计
- 交付数量统计图
- 产值统计图
- 资金占用统计图

**修改文件**:
- `Common/Dto/Dashboard/DashboardSummaryDto.java` — 添加新统计字段
- `Common/Dto/Dashboard/OrderStatsDto.java` — 新建订单统计 DTO
- `Common/Dto/Dashboard/FinancialStatsDto.java` — 新建财务统计 DTO
- `Common/Dto/Dashboard/ChartStatsDto.java` — 新建图表统计 DTO
- `Controller/DashboardController.java` — 添加统计接口

#### 4. 首页 - 订单追踪看板
**缺少**: 完整功能

**需要新建**:
- `Common/Dto/Dashboard/OrderTrackingDto.java` — 订单追踪 DTO
- `Controller/DashboardController.java` — 添加订单追踪看板接口

#### 5. 首页 - 待办事项
**已有**: 维保提醒通过 EquipmentMaintenanceController /reminder 实现

**缺少**:
- 环境预警汇总（如温湿度超标）
- 人员预警汇总（如证书到期、健康证到期）
- 统一的待办事项接口

**修改文件**:
- `Common/Dto/Dashboard/TodoStatsDto.java` — 新建待办统计 DTO
- `Controller/DashboardController.java` — 添加待办事项接口

#### 6. 人员档案 - 证书上传
**已有**: PersonnelFile 有 health_cert_no, health_cert_issue, health_cert_expire 字段
**已有**: 健康证到期提醒接口 /expiring

**缺少**:
- 证书文件上传（关联 FileInfo）
- 高级模糊查询（按人员名称、部门、岗位、身份证、入职时间）

**修改文件**:
- `Controller/PersonnelFileController.java` — 添加高级搜索接口 + 证书上传接口
- `Service/PersonnelFileService.java` — 添加高级搜索方法
- `Service/impl/PersonnelFileServiceImpl.java` — 实现
