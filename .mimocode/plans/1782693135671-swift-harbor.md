# Controller名称字段填充修复计划

## 目标
修复所有Controller，使其在返回Entity数据前自动填充关联表的名称字段。

## 修复方案

### 每个Controller需修改的内容

1. **添加依赖注入**：注入关联表的Service
2. **添加fillNameFields方法**：查询关联表并填充名称字段
3. **在查询方法中调用**：getById、list、search等返回数据前调用fillNameFields

### 示例代码模板

```java
// 1. 注入Service
@Autowired
private XxxService xxxService;

// 2. 添加fillNameFields方法
private void fillNameFields(Entity entity) {
    if (entity == null) return;
    if (entity.getXxxId() != null) {
        Xxx xxx = xxxService.getById(entity.getXxxId());
        if (xxx != null) {
            entity.setXxxName(xxx.getXxxName());
        }
    }
}

private void fillNameFieldsForList(List<Entity> list) {
    if (list == null) return;
    for (Entity entity : list) {
        fillNameFields(entity);
    }
}

// 3. 在返回数据前调用
@GetMapping("/{id}")
public ApiResponse<Entity> getById(@PathVariable Long id) {
    Entity entity = service.getById(id);
    fillNameFields(entity);  // 添加这行
    return success(entity);
}
```
