# 统一所有 Entity 审计字段

## 目标
所有 Entity 统一使用 4 个审计字段：`createdBy`、`updatedBy`、`createdTime`、`updatedTime`，数据库列名统一为 `created_by`、`updated_by`、`created_time`、`updated_time`，全部支持 MyBatis-Plus 自动填充。

## 标准字段定义（AuditEntity）
| Java 字段 | 类型 | DB 列名 | Fill 策略 |
|---|---|---|---|
| `createdBy` | Long | `created_by` | `FieldFill.INSERT` |
| `updatedBy` | Long | `updated_by` | `FieldFill.INSERT_UPDATE` |
| `createdTime` | LocalDateTime | `created_time` | `FieldFill.INSERT` |
| `updatedTime` | LocalDateTime | `updated_time` | `FieldFill.INSERT_UPDATE` |

## 执行顺序
1. 先执行数据库迁移脚本
2. 修改所有 Entity（继承 AuditEntity，删除手动字段）

4. 修改 XML Mapper
