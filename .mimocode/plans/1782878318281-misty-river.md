# 修复所有 XML Mapper 审计字段问题

## 问题统计
- **30 个 XML 文件**有问题（共 44 个）
- **17 个**属性名错误（createdAt/updatedAt/creatorId/updaterId）
- **22 个**列名错误（created_at/updated_at/creator_id/updater_id）
- **3 个**主键名错误（UserRole/UserDepartment/RolePerm）
- **1 个**幽灵字段（ProductionProcessRecord 的 updatedName）
- **4 个**完全缺少审计字段映射

## 修复步骤

1. 对每个 XML 文件：
   - 修改 ResultMap 中的 property 和 column 名称
   - 修改 Base_Column_List 中的列名
   - 补充缺失的字段映射
   - 修复主键名

2. 修改 DosageForm.java：删除 version 字段（数据库没有）

3. 编译验证：`./mvnw compile`
