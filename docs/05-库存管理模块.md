# 库存管理模块

## 模块概述

库存管理模块是ERP系统的核心模块之一，负责管理企业的物料和制剂库存。该模块实现了库存的入库、出库、库存查询、库存预警等功能，支持多仓库、多批次、多单位的库存管理，并提供了完整的库存交易记录用于审计和追溯。

## 功能特性

- 库存管理：库存查询、库存预警、库存调整
- 入库管理：入库单创建、入库明细管理、入库确认
- 出库管理：出库单创建、出库明细管理、出库确认
- 库存交易记录：完整的库存变动记录，用于审计和追溯
- 多仓库管理：支持多个生产单位的库存管理
- 批次管理：支持批次号、生产日期、有效期管理
- 库存预警：最低库存预警、最高库存限制
- 单号自动生成：入库单号、出库单号自动生成

**控制器**: [StockController.java](../src/main/java/com/tonghui/erp/Controller/StockController.java)

- **路径**: `GET /api/stock/search`

## 业务流程

### 入库流程

1. 创建入库单（自动生成入库单号）
2. 添加入库明细
3. 确认入库
4. 更新库存
5. 生成库存交易记录
6. 入库单状态变更为已完成

### 出库流程

1. 创建出库单（自动生成出库单号）
2. 添加出库明细
3. 检查库存是否充足
4. 确认出库
5. 更新库存
6. 生成库存交易记录
7. 出库单状态变更为已完成

### 库存预警流程

1. 系统定期检查库存
2. 当库存低于最低库存预警数量时，触发预警
3. 当库存高于最高库存限制数量时，触发预警
4. 通知相关人员进行处理

## 单据状态

- **草稿**: 单据创建后的初始状态
- **已确认**: 单据已确认，等待执行
- **已完成**: 单据已执行完成
- **已取消**: 单据被取消

## 业务规则

### 单号生成规则

1. 入库单号格式：`IN + 年月日 + 4位序列号`
2. 出库单号格式：`OUT + 年月日 + 4位序列号`
3. 每天序列号从0001开始
4. 系统自动生成，保证唯一性

### 库存管理规则

1. 入库时增加库存，出库时减少库存
2. 出库前必须检查库存是否充足
3. 库存低于最低预警数量时触发预警
4. 库存高于最高限制数量时触发预警
5. 支持批次管理，按批次出库

### 库存交易记录规则

1. 每次库存变动都生成交易记录
2. 交易记录包含变动前后的数量
3. 交易记录不可删除，用于审计和追溯

## 使用示例

### 创建入库单

```json
POST /api/stockin/withDetails
{
  "inCode": "IN20260205001",
  "inType": "采购入库",
  "prodUnitId": 1,
  "supplierId": 1,
  "relatedOrder": "PO20260205001",
  "inDate": "2026-02-05",
  "inStatus": "草稿",
  "remark": "采购入库"
}
```

### 创建出库单

```json
POST /api/stockout/withDetails
{
  "outCode": "OUT20260205001",
  "outType": "生产领用",
  "prodUnitId": 1,
  "relatedOrder": "WO20260205001",
  "outDate": "2026-02-05",
  "outStatus": "草稿",
  "remark": "生产领用"
}
```

### 查询库存

```http
GET /api/stock/search?pageIndex=0&pageSize=20&itemName=葡萄糖
```

### 生成入库单号

```http
GET /api/stockin/generateCode
```

## 扩展功能

### 库存盘点

实现定期库存盘点功能，对比实际库存和系统库存，生成盘点差异报告。

### 库存调拨

实现仓库间的库存调拨功能，支持跨仓库的库存转移。

### 库存预警通知

实现库存预警的自动通知功能，通过邮件、短信等方式通知相关人员。

### 库存分析报表

生成库存分析报表，包括库存周转率、库存结构分析、库存成本分析等。

## 注意事项

1. 入库单号和出库单号一旦生成，不建议修改
2. 删除入库单或出库单前，应检查单据状态
3. 出库前必须检查库存是否充足
4. 库存交易记录不可删除，用于审计和追溯
5. 批次管理时，应优先出库较早批次的物料（先进先出）
6. 有效期管理时，应优先出库即将过期的物料
