# ERP系统数据库设计文档 v2

> 本文档基于 `db/schema_v2.sql` 生成，共41张表，已修复v1版本所有已知问题。
> 数据库：MySQL 8.0+ | 引擎：InnoDB | 字符集：utf8mb4

---

## 全局约定

| 约定 | 说明 |
|------|------|
| 主键类型 | 统一为 `BIGINT AUTO_INCREMENT` |
| 软删除 | 所有业务表新增 `is_deleted TINYINT(1) DEFAULT 0` |
| 乐观锁 | 高并发表新增 `version INT DEFAULT 0` |
| 时间字段 | 统一为 `created_at` / `updated_at` |
| 外键关联 | 所有字符串替代外键的字段改为正式外键 |
| 冗余字段 | 有外键的情况下删除冗余字符串字段 |
