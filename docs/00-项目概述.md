# ERP系统项目概述

## 项目简介

本系统是一个完整的企业资源规划（ERP）系统，基于Spring Boot 3.5.5构建，采用Java 21开发。系统涵盖了用户权限管理、物料管理、生产管理、采购管理、库存管理、审批流程、文件管理等核心业务模块。

## 技术栈

### 后端框架
- **Spring Boot**: 3.5.5
- **Java**: 21
- **MyBatis-Plus**: 3.5.5
- **Spring Security**: 用于安全认证和授权

### 数据库
- **MySQL**: 8.0.33
- **JDBC**: MySQL Connector Java

### 安全与认证
- **JWT (JSON Web Token)**: 0.11.5 - 用于用户认证
- **Argon2**: 2.11 - 用于密码加密
- **Bouncy Castle**: 1.78.1 - 安全加密支持

### 工具库
- **Lombok**: 简化Java代码
- **Apache POI**: 5.2.3 - Excel文件操作

## 架构设计

### 分层架构
系统采用经典的三层架构：
- **Controller层**: 处理HTTP请求，参数验证，调用Service层
- **Service层**: 业务逻辑处理，事务管理
- **Data层**: 数据访问，使用MyBatis-Plus进行ORM映射

### 安全机制
- **JWT认证**: 基于令牌的无状态认证
- **密码加密**: 使用Argon2算法进行密码哈希
- **权限控制**: 基于角色和权限的访问控制（RBAC）
- **请求拦截**: JWT拦截器验证所有API请求

### 数据访问
- **MyBatis-Plus**: 简化CRUD操作
- **分页支持**: 内置分页功能
- **动态SQL**: XML映射文件支持复杂查询

## 启动说明

### 环境要求
- JDK 21+
- Maven 3.6+
- MySQL 8.0+

### 启动步骤
1. 配置数据库连接信息（application.yml）
2. 执行数据库初始化脚本
3. 运行 `mvn spring-boot:run` 或直接运行 `ErpApplication.java`
4. 访问API接口，默认端口为8080

### 默认账户
系统会自动初始化root用户，具体信息请查看 `RootUserInitializer.java`

## API接口规范

### 认证方式
所有API接口（除登录接口外）需要在请求头中携带JWT令牌：
```
Authorization: Bearer <token>
```

### 响应格式
统一使用 `ApiResponse` 格式：
```json
{
  "success": true,
  "message": "操作成功",
  "data": {},
  "code": 200
}
```

### 分页请求
使用 `PageRequestDto` 进行分页：
```json
{
  "pageIndex": 0,
  "pageSize": 10
}
```

## 开发规范

### 代码规范
- 使用Lombok简化实体类
- 遵循RESTful API设计原则
- 统一异常处理
- 统一日志记录

### 命名规范
- 数据库表名：小写+下划线（如：user_account）
- Java类名：大驼峰（如：UserAccount）
- 接口方法：小驼峰（如：getUserById）
