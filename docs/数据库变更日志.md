## v3→v4 变更（2024-06-29）

### 新增表（1张）

| 表名 | 说明 |
|------|------|
| `business_file` | 业务文件关联表，统一管理所有业务表与文件的关联 |

### 修改表

| 表名 | 变更内容 |
|------|----------|
| `personnel_file` | 删除health_cert_file_id字段，改用business_file关联 |
| `preparation_document` | 删除file_id字段，改用business_file关联 |

### business_file 使用方式

```sql
-- 查询某制剂的所有文件
SELECT * FROM business_file 
WHERE business_type = 'PREPARATION' AND business_id = 123;

-- 查询某人员的健康证
SELECT f.* FROM file_info f 
JOIN business_file bf ON f.file_id = bf.file_id 
WHERE bf.business_type = 'PERSONNEL_FILE' 
  AND bf.business_id = 456 
  AND bf.file_purpose = 'HEALTH_CERT';

-- 关联业务类型枚举
-- USER / PERSONNEL_FILE / PREPARATION / EQUIPMENT / PURCHASE_ORDER / STOCK_IN / STOCK_OUT / APPROVAL

-- 文件用途枚举
-- HEALTH_CERT / PROCESS_SPEC / VALIDATION_PLAN / VALIDATION_REPORT / BATCH_RECORD / ATTACHMENT / IMAGE
```
